# StudyFlow

StudyFlow is a student productivity web application designed to help secondary school and university students organize their academic life.

## ✨ Features (Planned)

- Task Management
- Subject Management
- Notes
- Dashboard
- Exam Countdown
- Study Planner

## 🛠️ Tech Stack

- React
- JavaScript (ES6+)
- Tilwindcss
- Vite

## 📅 Current Status

🚧 In Development (Phase 1)

## 🎯 Goal

To build a scalable productivity application while learning modern React development.
